# ⚓ Hormuz

A river navigation game built in pure HTML5/Canvas — no external dependencies required.

## 🎮 Play

Simply open `index.html` in any modern browser.  

## 🕹️ Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move | Arrow keys / WASD | Drag finger or Joystick |
| Machine gun fire | Space | 🔫 Button |
| Shield | X | 🛡 Button |

## 📋 Rules

- Collect **drowning chicks** to score points
  - 🐥 Small = 1 pt · 🐤 Medium = 2 pts · 🐣 Large = 3 pts
- Each chick collected adds **+10 seconds** to the timer
- **30 points** = advance to next level

## 🗺️ Level Progression

| Level | New feature |
|-------|-------------|
| 1–3 | Rocks, logs, crocodiles |
| 2+ | Floating mines |
| 4+ | Pirate ships |
| 5+ | Whirlpools (suck obstacles in!) |
| 6+ | Armed pirates + Machine gun + Shield |
| 7+ | 🍬 Immunity candy · 🦈 Sharks |
| 8+ | 🏝 Central islands blocking passage |
| 9+ | Narrow passage zones |
| 10+ | Smart homing pirate ships |
| 11+ | 💀 SUDDEN DEATH — unlimited, 2× pirate boats |
